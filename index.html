제품 요구사항 정의서 (PRD)

고등학생 포트폴리오 웹사이트

Who — 누가

대상 사용자

  • 제작자: 대입 또는 취업을 준비하는 고등학생 (포트폴리오 주인)
  • 열람자: 대학 입학사정관, 기업 채용 담당자, 교사, 지인
  • 관리자: 사이트 오너(학생) — 관리자 계정으로 콘텐츠 직접 수정
What — 무엇을

제품 개요

자기소개, 교내외 활동, 프로젝트 경험, 갤러리, 영상, 연락처를 담은 1페이지 스크롤형 개인 포트폴리오 웹사이트. 관리자 로그인 후 텍스트·이미지·영상을 직접 편집 가능.

When — 언제

사용 시점

  • 입시·취업 원서 제출 전 자기 어필 자료로 공유
  • 면접 전 면접관이 사전에 열람
  • 활동·프로젝트 완료 즉시 관리자 모드로 업데이트
  • 포트폴리오 URL을 자소서·이메일에 첨부하는 시점
Where — 어디서

사용 환경

  • PC 브라우저 (크롬, 사파리, 엣지) — 주 열람 환경
  • 모바일·태블릿 — 반응형 레이아웃 지원
  • 웹 호스팅 서버 (GitHub Pages, Netlify 등)에 배포 가능한 단일 HTML 파일
How — 어떻게

핵심 기능 목록

  • 자기소개: 프로필 사진, 기본 정보(학교·학년·진로), 소개 텍스트, 해시태그
  • 교내외 활동: 타임라인 형식, 연도·기관·설명 관리
  • 프로젝트 경험: 카드 형식, 이미지·유형·설명·태그 포함
  • 작품 갤러리: 그리드 형식, 이미지 다중 업로드
  • 영상 자료: YouTube URL 삽입 또는 파일 직접 업로드
  • 연락처: 이메일·전화·SNS 링크, 메시지 폼
  • 관리자 로그인: ID qwer1234 / PW qwer1234, 로그인 후 인라인 편집 모드 활성화
  • 콘텐츠 편집: 텍스트 클릭 후 직접 수정, 이미지 클릭 후 파일 교체
Why — 왜

제작 목적 및 기대 효과

  • 서류만으로 전달하기 어려운 역량·개성을 시각적으로 표현
  • 입학사정관·채용 담당자에게 한눈에 파악 가능한 종합 포트폴리오 제공
  • 언제든 직접 수정 가능하여 최신 이력·활동을 실시간 반영
  • 별도 코딩 없이 관리자 로그인만으로 콘텐츠 관리 가능
